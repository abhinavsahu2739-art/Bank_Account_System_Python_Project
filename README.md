# Bank Account System Python Project

A simple command-line banking application built using Python and Object-Oriented Programming concepts. This project allows a user to deposit money, withdraw money, and check the current account balance through a menu-driven interface.

## Project Overview

This project is designed for academic learning and presentation purposes. It demonstrates the basic working of a banking system without using any database. All data is stored temporarily during program execution.

## Features

- Deposit money into the account
- Withdraw money from the account
- Check current balance
- Input validation using exception handling
- Menu-driven command-line interface
- Simple and beginner-friendly code structure

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## Concepts Used

- Class and Object
- Constructor (`__init__`)
- Instance variables
- Methods
- Conditional statements
- Loops
- Exception handling
